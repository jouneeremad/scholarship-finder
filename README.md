# Scholarship Finder

A Python application that helps students find and organize scholarship opportunities.

## Features

- Add scholarships
- Search scholarships
- Filter by country
- Save scholarships automatically
- Add scholarships to favorites
- Save favorites automatically

## Technologies Used

- Python
- File Handling

## Future Improvements

- Scholarship recommendations
- AI matching system
- Scholarship ranking
- Deadline reminders
- Web application version
